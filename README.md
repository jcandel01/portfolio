# Jaime Candel — Portfolio

My personal portfolio built with **React + Vite + TypeScript + Tailwind CSS** and **Framer Motion**.

Features interactive in-browser demos of my projects (Sphere, best365, EcoVecinos, IdeaTracker, Storyforge) rendered inside phone and browser frames.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Includes `netlify.toml` configured for Netlify. Push to GitHub and import from the Netlify dashboard, or run:

```bash
npx netlify deploy --build
```
